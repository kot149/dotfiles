# ファイル・バッファ操作

## 基本的なファイル操作

| キー | 動作 | VSCode相当 |
|------|------|-----------|
| `Ctrl+S` | 保存 | Ctrl+S |
| `Ctrl+W` | 現在のバッファを閉じる | Ctrl+W |
| `Ctrl+P` | ファイルをファジー検索して開く | Ctrl+P |

## バッファとは

vimでは開いているファイルを **バッファ** と呼びます。VSCodeの「タブ」に相当します。

```
Ctrl+P でファイルを開く → バッファが増える
Ctrl+W で閉じる         → バッファが減る
Ctrl+Tab で切り替える   → 別のバッファへ移動
```

## バッファの切り替え

| キー | 動作 |
|------|------|
| `Ctrl+Tab` | 次のバッファへ |
| `Ctrl+Shift+Tab` | 前のバッファへ |

## ファイルエクスプローラ (Neo-tree)

| キー | 動作 |
|------|------|
| `Ctrl+B` | エクスプローラを開閉 |
| `Ctrl+Shift+E` | エクスプローラにフォーカス |

### エクスプローラ内の操作

エクスプローラにフォーカスしている状態で使えるキー:

| キー | 動作 |
|------|------|
| `Enter` | ファイルを開く / フォルダを展開 |
| `a` | 新しいファイルを作成 |
| `A` | 新しいフォルダを作成 |
| `r` | ファイル/フォルダをリネーム |
| `d` | ファイル/フォルダを削除 |
| `y` | ファイルをコピー |
| `x` | ファイルを切り取り |
| `p` | 貼り付け |
| `q` | エクスプローラを閉じる |
| `?` | ヘルプを表示 |

## ウィンドウ（分割）操作

vimでは複数のウィンドウ（分割画面）を同時に開けます。

| キー | 動作 | VSCode相当 |
|------|------|-----------|
| `Ctrl+\` | 右に分割（縦分割） | Ctrl+\ |
| `:split` | 下に分割（横分割） | — |
| `Ctrl+h/j/k/l` | ウィンドウ間を移動 | — |
| `:close` | 現在のウィンドウを閉じる | — |
| `:only` | 現在のウィンドウ以外を全て閉じる | — |

## ターミナル

| キー | 動作 |
|------|------|
| `Ctrl+\`` | ターミナルを開閉 |
| `Esc` | ターミナルモードから抜ける |

ターミナルを開いた後、コマンドを入力するには自動的に挿入モードになります。  
`Esc` を押すとノーマルモードに戻り、ターミナルの出力をスクロールしたりコピーできます。

## コマンドラインからの操作

ノーマルモードで `:` を押してコマンドを入力します。

| コマンド | 動作 |
|----------|------|
| `:w` | 保存 |
| `:w ファイル名` | 別名で保存 |
| `:q` | 終了（変更がある場合は失敗） |
| `:wq` または `:x` | 保存して終了 |
| `:q!` | 保存せずに強制終了 |
| `:e ファイルパス` | ファイルを開く |
| `:e!` | ファイルを再読み込み（変更を破棄） |
| `:bd` | バッファを閉じる |

## プラグイン管理

| コマンド | 動作 |
|----------|------|
| `:Lazy` | プラグイン管理UIを開く |
| `:Mason` | LSPサーバー管理UIを開く |
