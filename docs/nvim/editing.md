# テキスト編集

## コピー・貼り付け・切り取り

vimではコピーを **ヤンク (yank)**、貼り付けを **プット (put)** と呼びます。

### ビジュアルモードで選択してから操作する

```
1. v（または V で行選択）を押してビジュアルモードへ
2. 矢印キーや移動コマンドで範囲を選択
3. 以下の操作キーを押す
```

| キー | 動作 | VSCode相当 |
|------|------|-----------|
| `y` | コピー (yank) | Ctrl+C |
| `d` | 切り取り | Ctrl+X |
| `p` | カーソルの後に貼り付け | Ctrl+V |
| `P` | カーソルの前に貼り付け | — |

> システムクリップボードと連携済みのため、VSCodeや他アプリとのコピペもそのまま使えます。

### ノーマルモードでの高速コピー・削除

| キー | 動作 |
|------|------|
| `yy` | 現在行をコピー |
| `dd` | 現在行を切り取り（削除） |
| `yw` | カーソルから次の単語末尾までコピー |
| `dw` | カーソルから次の単語末尾まで削除 |
| `y$` | カーソルから行末までコピー |
| `d$` | カーソルから行末まで削除 |

## 行の移動・コピー（VSCode風）

| キー | 動作 |
|------|------|
| `Alt+↑` | 行を上に移動 |
| `Alt+↓` | 行を下に移動 |
| `Alt+Shift+↑` | 行を上にコピー |
| `Alt+Shift+↓` | 行を下にコピー |

ビジュアルモードで複数行を選択した状態でも使えます。

## 削除・変更

| キー | 動作 |
|------|------|
| `x` | カーソル上の1文字を削除 |
| `X` | カーソルの前の1文字を削除 |
| `dd` | 現在行を削除 |
| `Ctrl+Shift+K` | 現在行を削除（VSCode風） |
| `r{文字}` | カーソル上の1文字を{文字}に置き換え |
| `cw` | カーソルから次の単語末尾まで削除して挿入モードへ |
| `cc` | 現在行を削除して挿入モードへ |
| `C` | カーソルから行末まで削除して挿入モードへ |

## アンドゥ・リドゥ

| キー | 動作 | VSCode相当 |
|------|------|-----------|
| `u` | アンドゥ | Ctrl+Z |
| `Ctrl+r` | リドゥ | Ctrl+Y |

vimのアンドゥは非常に強力で、ファイルを閉じた後も履歴が残ります（`undofile`設定済み）。

## インデント

| キー | 動作 |
|------|------|
| `>>` | 現在行を右にインデント |
| `<<` | 現在行を左にインデント |
| ビジュアル選択 → `Tab` | 選択範囲を右にインデント |
| ビジュアル選択 → `Shift+Tab` | 選択範囲を左にインデント |

## コメントトグル

| キー | 動作 |
|------|------|
| `Ctrl+/` | 現在行のコメントをトグル |
| ビジュアル選択 → `Ctrl+/` | 選択範囲のコメントをトグル |

> ターミナルによっては `Ctrl+/` が届かず `Ctrl+_` にフォールバックします。どちらも同じ動作をします。

## 繰り返し

| キー | 動作 |
|------|------|
| `.` | 直前の操作を繰り返す |
| `{数字}{コマンド}` | コマンドを{数字}回繰り返す（例: `3dd` で3行削除） |

`.`（ドット）は非常に便利なコマンドです。同じ操作を複数箇所に適用するときに使います。

## テキストオブジェクト（上級）

vimには「単語の中」「括弧の中」などを1コマンドで選択・操作できる仕組みがあります。

```
{操作}{i または a}{対象}

i = inner（対象の内側のみ）
a = around（対象を含む）
```

| コマンド | 動作 |
|---------|------|
| `viw` | カーソル上の単語を選択 |
| `vaw` | カーソル上の単語を空白込みで選択 |
| `vi"` | `"..."` の中を選択 |
| `va"` | `"..."` ごと選択 |
| `vi(` | `(...)` の中を選択 |
| `vi{` | `{...}` の中を選択 |
| `ci"` | `"..."` の中を削除して挿入モードへ |
| `di(` | `(...)` の中を削除 |

例: `console.log("hello")` で `ci"` を押すと `"hello"` の中身が消えて挿入モードになります。
