# キーバインド一覧

`n` = ノーマルモード、`i` = 挿入モード、`v` = ビジュアルモード、`t` = ターミナルモード

## ファイル・バッファ

| キー | モード | 動作 |
|------|--------|------|
| `Ctrl+S` | n/i/v | 保存 |
| `Ctrl+W` | n | バッファを閉じる |
| `Ctrl+P` | n | Quick Open（ファイル検索） |
| `Ctrl+Shift+P` | n | コマンドパレット |
| `Ctrl+Tab` | n | 次のバッファへ |
| `Ctrl+Shift+Tab` | n | 前のバッファへ |

## 検索

| キー | モード | 動作 |
|------|--------|------|
| `Ctrl+F` | n | ファイル内検索 |
| `Ctrl+H` | n | 検索と置換 |
| `Ctrl+Shift+F` | n | ワークスペース全体検索 |
| `n` / `N` | n | 次/前の検索一致へ |
| `*` | n | カーソル下の単語を前方検索 |
| `#` | n | カーソル下の単語を後方検索 |
| `Esc` | n | 検索ハイライトを消す |

## エクスプローラ・パネル

| キー | モード | 動作 |
|------|--------|------|
| `Ctrl+B` | n | エクスプローラをトグル |
| `Ctrl+Shift+E` | n | エクスプローラにフォーカス |
| `Ctrl+\`` | n/t | ターミナルをトグル |
| `Ctrl+Shift+M` | n | Problems パネルをトグル |

## 編集

| キー | モード | 動作 |
|------|--------|------|
| `Ctrl+/` | n/i/v | コメントをトグル |
| `Alt+↑` | n/i/v | 行を上に移動 |
| `Alt+↓` | n/i/v | 行を下に移動 |
| `Alt+Shift+↑` | n/i/v | 行を上にコピー |
| `Alt+Shift+↓` | n/i/v | 行を下にコピー |
| `Ctrl+Shift+K` | n/i | 行を削除 |
| `Ctrl+Enter` | n/i | 下に行を挿入 |
| `Ctrl+Shift+Enter` | n/i | 上に行を挿入 |
| `Tab` | v | インデント |
| `Shift+Tab` | v | アウトデント |
| `u` | n | アンドゥ |
| `Ctrl+R` | n | リドゥ |

## LSP・コード

| キー | モード | 動作 |
|------|--------|------|
| `F12` | n | 定義へジャンプ |
| `Ctrl+O` | n | ジャンプ前の位置に戻る |
| `F2` | n | シンボルのリネーム |
| `K` | n | ホバードキュメント表示 |
| `Shift+Alt+F` | n | ドキュメントをフォーマット |
| `Ctrl+Shift+.` | n | コードアクション |
| `Space+.` | n/v | コードアクション（代替） |

## 補完

| キー | モード | 動作 |
|------|--------|------|
| `Ctrl+Space` | i | 補完を明示的に起動 |
| `Tab` / `Shift+Tab` | i | 候補リストの上下移動 |
| `Ctrl+N` / `Ctrl+P` | i | 候補リストの上下移動（代替） |
| `Enter` | i | 候補を確定 |
| `Ctrl+E` | i | 補完ウィンドウを閉じる |

## ウィンドウ分割

| キー | モード | 動作 |
|------|--------|------|
| `Ctrl+\` | n | 右に分割（縦分割） |
| `Ctrl+H` | n | 左のウィンドウへ移動 |
| `Ctrl+J` | n | 下のウィンドウへ移動 |
| `Ctrl+K` | n | 上のウィンドウへ移動 |
| `Ctrl+L` | n | 右のウィンドウへ移動 |

## モード切り替え

| キー | モード | 動作 |
|------|--------|------|
| `i` | n | 挿入モードへ（カーソル左） |
| `a` | n | 挿入モードへ（カーソル右） |
| `o` | n | 下に行を作って挿入モードへ |
| `O` | n | 上に行を作って挿入モードへ |
| `v` | n | ビジュアルモードへ（文字選択） |
| `V` | n | ビジュアルモードへ（行選択） |
| `Ctrl+V` | n | ビジュアルモードへ（矩形選択） |
| `Esc` | i/v | ノーマルモードに戻る |

## ナビゲーション

| キー | モード | 動作 |
|------|--------|------|
| `←` `↑` `↓` `→` | n/i/v | 1文字/1行移動 |
| `Ctrl+←` `Ctrl+→` | n/i | 単語単位で移動 |
| `Home` / `End` | n/i | 行頭 / 行末 |
| `Ctrl+Home` | n/i | ファイル先頭 |
| `Ctrl+End` | n/i | ファイル末尾 |
| `Ctrl+G` | n | 指定行へジャンプ |
| `gg` | n | ファイルの先頭 |
| `G` | n | ファイルの末尾 |
| `{数字}G` | n | {数字}行目へ |
| `w` / `b` | n | 次/前の単語へ |
| `%` | n | 対応する括弧へ |
| `Ctrl+U` / `Ctrl+D` | n | 半ページ上/下 |

## Git (gitsigns)

| キー | モード | 動作 |
|------|--------|------|
| `Space+gs` | n | ハンクをステージ |
| `Space+gr` | n | ハンクをリセット |
| `Space+gd` | n | 差分を表示 |
| `Space+gb` | n | 行の blame を表示 |

## その他

| キー | モード | 動作 |
|------|--------|------|
| `Space+s` | n | Flash ジャンプ（高速移動） |
| `.` | n | 直前の操作を繰り返す |
| `Esc` | t | ターミナルモードを抜ける |
