# カーソル移動

この設定では矢印キーが使えるため、VSCodeと同じ感覚で移動できます。  
加えて、vimの高速移動コマンドを覚えると編集効率が大きく上がります。

## 基本移動（VSCodeと同じ）

| キー | 動作 |
|------|------|
| `←` `↑` `↓` `→` | 1文字/1行移動 |
| `Ctrl+←` `Ctrl+→` | 単語単位で移動 |
| `Home` / `End` | 行頭 / 行末 |
| `Ctrl+Home` | ファイルの先頭 |
| `Ctrl+End` | ファイルの末尾 |
| `PgUp` / `PgDn` | ページ単位でスクロール |
| `Ctrl+G` | 指定した行番号へジャンプ |

## vim独自の移動（ノーマルモード）

慣れると矢印キーより速く移動できます。

### 文字・単語単位

| キー | 動作 |
|------|------|
| `h` `j` `k` `l` | 左・下・上・右（矢印キーと同じ） |
| `w` | 次の単語の先頭へ |
| `W` | 次の単語の先頭へ（記号を単語区切りとしない） |
| `b` | 前の単語の先頭へ |
| `e` | 次の単語の末尾へ |
| `0` | 行の絶対先頭（インデントを含む） |
| `^` | 行の最初の文字（インデントを除く） |
| `$` | 行末 |

### 行・ページ単位

| キー | 動作 |
|------|------|
| `gg` | ファイルの先頭 |
| `G` | ファイルの末尾 |
| `{数字}G` | {数字}行目へ（例: `42G` で42行目） |
| `{` | 前の空行（段落の先頭）へ |
| `}` | 次の空行（段落の末尾）へ |
| `Ctrl+u` | 半ページ上スクロール |
| `Ctrl+d` | 半ページ下スクロール |
| `Ctrl+f` | 1ページ下スクロール |
| `Ctrl+b` | 1ページ上スクロール |

### 画面内移動

| キー | 動作 |
|------|------|
| `H` | 画面上端の行へ |
| `M` | 画面中央の行へ |
| `L` | 画面下端の行へ |
| `zz` | カーソル行を画面中央に表示 |
| `zt` | カーソル行を画面上端に表示 |
| `zb` | カーソル行を画面下端に表示 |

### 対応する記号へジャンプ

| キー | 動作 |
|------|------|
| `%` | 対応する括弧へ（`()`、`[]`、`{}`） |
| `f{文字}` | 行内の次の{文字}へ（例: `f,` で次の `,` へ） |
| `F{文字}` | 行内の前の{文字}へ |
| `;` | 直前の `f`/`F` 検索を繰り返す |

## ジャンプ履歴

| キー | 動作 |
|------|------|
| `Ctrl+o` | 直前のジャンプ位置に戻る（VSCodeの「戻る」相当） |
| `Ctrl+i` | 次のジャンプ位置へ進む（VSCodeの「進む」相当） |

`F12`（定義へジャンプ）後に `Ctrl+o` で元の場所に戻れます。

## ウィンドウ間の移動

| キー | 動作 |
|------|------|
| `Ctrl+h` | 左のウィンドウへ |
| `Ctrl+j` | 下のウィンドウへ |
| `Ctrl+k` | 上のウィンドウへ |
| `Ctrl+l` | 右のウィンドウへ |
