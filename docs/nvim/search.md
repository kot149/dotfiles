# 検索・置換

## ファイル内検索

| キー | 動作 |
|------|------|
| `Ctrl+F` | 検索ワードの入力を開始 |
| `n` | 次の一致箇所へ |
| `N` | 前の一致箇所へ |
| `Esc` | 検索ハイライトを消す |

検索は大文字・小文字を区別しませんが（`ignorecase`設定）、  
大文字を1文字でも含む検索ワードは区別されます（`smartcase`設定）。

### 検索パターン（vimの正規表現）

| パターン | 意味 | 例 |
|---------|------|---|
| `.` | 任意の1文字 | `f.o` → foo, f1o |
| `*` | 直前の文字の0回以上繰り返し | `fo*` → f, fo, foo |
| `^` | 行頭 | `^def` → 行頭の def |
| `$` | 行末 | `end$` → 行末の end |
| `\w` | 単語文字 (英数字+アンダースコア) | |
| `\s` | 空白文字 | |

## ファイル内置換

ノーマルモードで `Ctrl+H` を押すと置換コマンドの入力が始まります。

```
:%s/検索ワード/置換後/g
```

| オプション | 意味 |
|-----------|------|
| `g` | 行内の全一致を置換（省略すると最初の1つだけ） |
| `i` | 大文字・小文字を区別しない |
| `c` | 1つずつ確認しながら置換 |

**例:**

```
:%s/foo/bar/g         ファイル全体の foo を bar に置換
:%s/foo/bar/gi        大文字小文字を区別せず置換
:%s/foo/bar/gc        確認しながら置換（y=置換、n=スキップ、q=中止）
```

ビジュアルモードで範囲を選択してから `:s/foo/bar/g` とすると、選択範囲のみ置換できます。

## ワークスペース全体検索 (Telescope)

| キー | 動作 |
|------|------|
| `Ctrl+Shift+F` | ワークスペース内のテキストを検索 |
| `Ctrl+P` | ファイル名で検索して開く |

### Telescope内の操作

検索ウィンドウが開いている状態での操作:

| キー | 動作 |
|------|------|
| 文字入力 | 検索ワードを絞り込む |
| `Ctrl+N` / `Ctrl+P` | 候補リストの上下移動 |
| `Enter` | 選択したファイル/箇所を開く |
| `Ctrl+X` | 横分割で開く |
| `Ctrl+V` または `Ctrl+X 3` | 縦分割で開く |
| `Esc` または `Ctrl+G` | 閉じる |

## カーソル下の単語を検索

ノーマルモードで以下のキーを押すと、カーソル下の単語をすぐに検索できます。

| キー | 動作 |
|------|------|
| `*` | カーソル下の単語を前方検索 |
| `#` | カーソル下の単語を後方検索 |
| `n` / `N` | 次/前の一致へ移動 |
